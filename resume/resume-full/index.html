    Personal details¶

    Name: Rene Peuser
    Year of birth: 1981
    Experience: More than 20 years
    Languages: German, English (fluent), Hungarian (basic)

    Professional summary¶

    Principal-level .NET software architect and framework engineer with a career spanning industrial automation, desktop architecture, developer tooling, APIs, cloud systems, and enterprise platforms.

    My central strength is identifying repeated engineering patterns and converting them into reusable SDKs, frameworks, test platforms, analyzers, templates, and automation.

    Experience¶

    Siemens · 2020–present¶

    • Reusable ASP.NET Core and Minimal API foundations
    • Capability meta-framework for a multi-provider cloud and AI platform
    • OpenAPI polymorphism, document transformations, schemas, and typed requests
    • Conditional, cross-property, collection, and domain validation
    • API integration and regression testing SDKs
    • Error handling and Problem Details
    • AWS, Azure, GCP, Snowflake, Fabric, OpenAI, and internal platform abstractions
    • DynamoDB, Lambda, Cognito, API Gateway, Step Functions, SSM
    • Architecture reviews, pair programming, coaching, and documentation

    SoftwareOne · 2021–present¶

    • Anti-corruption-layer APIs for Navision and Dynamics 365
    • Stage-switchable integration-test environment
    • CQRS with MediatR
    • Azure, Azure DevOps, Terraform, Application Insights, Grafana
    • Async and parallel performance improvements
    • Technical documentation and training

    TRUMPF Laser- und Systemtechnik · 2017–2020¶

    • WPF HMI features for industrial systems

    • .NET global tools

    • Performance and memory analysis
    • Automated tests and reusable development tooling

    Open source¶

    Nine public packages with more than 4.1 million cumulative downloads as of July 2026. Key packages include Argument.Check, Extensions.Pack, AspNetCore.Simple.MsTest.Sdk, AspNetCore.Simple.Sdk, and Solution.Parser.
