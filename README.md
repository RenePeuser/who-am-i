## What I build

- Enterprise-grade ASP.NET Core and Minimal API platforms
- Metadata-driven capability frameworks
- Reusable validation and error-handling infrastructure
- OpenAPI and schema-generation tooling
- Integration, contract, snapshot, and API testing SDKs
- Roslyn-based code analysis and developer tooling
- Cloud-native backend foundations for AWS, Azure, and GCP
- NuGet packages, CLI tools, templates, and automation
