# Contributing

This is a personal engineering portfolio. Corrections, broken-link reports, and factual clarifications are welcome through GitHub issues.

## Content rules

- Do not add confidential customer or employer information.
- Do not invent metrics, responsibilities, or achievements.
- Prefer precise technical language over marketing claims.
- Mark uncertain statements with `TODO: verify`.
- Keep public package statistics time-stamped and linked to their live source.
- Use Conventional Commits.

## Local preview

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

```
