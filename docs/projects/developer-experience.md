# Developer experience

Developer experience is where architecture becomes daily reality.

## Tooling themes

- NuGet-based reusable SDKs
- .NET global tools
- CLI filtering and automation
- Visual Studio extensions
- Roslyn analyzers
- templates and project foundations
- code-generation and schema-generation workflows
- test clients and assertion DSLs
- technical documentation and examples
- build optimization and pipeline checks

## Historical examples

- Visual Studio extensions that generated WPF view tests
- localization tooling for RESX and ResourceDictionary workflows
- Roslyn-based architecture rules
- solution and project setting verification
- CLI tools for repetitive engineering tasks
- reusable API and test templates

## Design test

A platform feature is incomplete until a developer can discover it, adopt it, diagnose it, test it, and migrate it.
