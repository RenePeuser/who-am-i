# Project portfolio

The projects in this portfolio are selected because they demonstrate reusable engineering rather than isolated features.

| Project | Primary engineering theme |
|---|---|
| Capability meta-framework | Metadata, extensibility, provider abstraction |
| Minimal API platform | Reusable API infrastructure |
| Testing SDK | Developer experience and regression safety |
| Validation engine | Semantic rules and cross-property constraints |
| OpenAPI infrastructure | Schemas, polymorphism, generated contracts |
| Error handling | Consistent Problem Details and diagnostics |
| Deployment/provider model | Multi-cloud lifecycle abstraction |
| Developer experience | CLI, analyzers, templates, automation |
