# TRUMPF

**Period:** 2015–2020

## Focus

Industrial HMI systems, WPF architecture, automated testing, developer tooling, performance engineering, and gradual adoption of ASP.NET Core.

## Contributions

- Features for touch-based high-tech laser-machine HMIs
- WPF and MVVM architecture
- Automated generation of view tests through Visual Studio extensions
- Roslyn analyzers enforcing architectural concepts
- Automated checks for solution and project settings
- Unit, integration, and system testing
- Performance and memory profiling
- .NET global tools
- Integration of simple Blender-generated 3D machine models into WPF
- Code reviews, refactoring, and testability improvements

## Technologies

C#, .NET Framework, .NET Core, WPF, MVVM, Prism, LINQ, TPL, ASP.NET Core, Roslyn, Visual Studio Extensions, MSTest, NSubstitute, SQL Server, Perforce, NuGet
