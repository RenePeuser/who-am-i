# Source and confidentiality policy

This portfolio combines:

- public package metadata;
- public conference and repository links;
- historical information from my resume;
- generalized descriptions of professional work.

## Confidentiality

Employer and customer work is described only at an architectural level. This repository must not contain:

- internal URLs or repository names;
- credentials or identifiers;
- customer data;
- proprietary algorithms;
- non-public source code;
- internal diagrams;
- confidential performance, cost, or business metrics.

## Statistics

Package counts, versions, and downloads are time-sensitive. Every statistic should include a date and link to the live source.

## Accuracy

Unverified claims should be marked `TODO: verify` rather than presented as fact.
