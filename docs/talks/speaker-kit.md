# Speaker kit

## Short bio

Rene Peuser is a Principal Software Architect, framework engineer, open-source maintainer, and conference speaker with more than 20 years of experience. He specializes in reusable .NET platforms, Minimal APIs, testing SDKs, OpenAPI, validation engines, and developer tooling.

## Long bio

Rene Peuser began his career in industrial automation and machine commissioning before moving into C#, WPF, MVVM, Visual Studio extensions, Roslyn, ASP.NET Core, and cloud platform engineering. Today he designs reusable frameworks and SDKs that standardize API infrastructure, testing, validation, OpenAPI, error handling, and multi-provider capability systems. His public NuGet portfolio has accumulated more than four million downloads.

## Suggested topics

- Building frameworks instead of repeating infrastructure
- Testing Minimal APIs at scale
- Designing a metadata-driven capability platform
- Conditional validation in C# attribute-based models
- OpenAPI polymorphism without losing maintainability
- Developer experience as an architectural concern
- From industrial automation to platform engineering
